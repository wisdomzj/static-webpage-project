台湾富瑞行销管理顾问有限公司

法定代表人： 许雪华

福州富瑞营销策划有限公司是由台湾富瑞行销管理顾问有限公司投资设立。该母公司为2001年创立于台湾台北，主要成员来自台湾及国际上市公司，专业涵盖企业管理、高端科技、市场营销、金融资本操作、文化创意....等多领域，与两岸政府、企业及民间团体关系良好，拥有丰沛的产官学界之人脉与资源。公司长年协助国内多个地方政府推动政策性两岸交流活动，并从事促进两岸有关各类产业发展、学术研究及媒体运作等共同合作的工作。2016年台湾富瑞营销管理顾问有限公司成为福建省知识产权智库战略合作机构，为有效配合政策及落实各项服务工作，遂成立福州富瑞营销策划有限公司，除了福州常驻人员，母公司之高管及专家团队将全力参与福建分公司运营，将台湾的各项专业技术及缜密的服务质量零时差的于福建省呈现。
